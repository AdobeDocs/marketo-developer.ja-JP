# パートナーサンドボックス

パートナーサンドボックスとは、ソフトウェア開発ライフサイクル中にパートナーが使用するMarketo Engage インスタンスです。 サンドボックスを利用して、Marketo Engage UIを探索し、Marketo Engage API サービスにアクセスできます。

Adobeは、パートナーが[Adobe Exchange パートナープログラム デジタルエクスペリエンス ](https://partners.adobe.com/technologyprogram/experiencecloud.html)に受け入れられた後、サンドボックスをプロビジョニングします。

ほとんどの点で、パートナーサンドボックスは機能的には実稼動インスタンスと同等です。 ただし、パートナーサンドボックスにはアドオンモジュールが含まれておらず、容量の制限も固定されています。

## 機能の概要

- サンドボックス：パートナーごとに1つ
- API呼び出し：1日あたり50,000
- リードデータベースレコード：10,000
- メール送信：キャンペーン実行ごとに30件
- カスタムオブジェクト：2つ
- ユーザー：25人
